◆ on_message_EDT_CGW_E_OTA_RESULT()

on_message_EDT_CGW_E_OTA_RESULT ( )

EDT에서 OTA 결과 메시지 수신 이벤트 핸들러

EDT로부터 받은 OTA 결과를 TMU에 중계하여 전달

  • 0x05: OTA 성공
  • 0xAA: 롤백 완료
  • 0xFF: OTA 실패

  • 0x06: 롤백 신호를 EDT에 전달
